---
title: 'Invite-Only Memberships'
---

<img src="/images/creating-campaign/membership/inviteonly/using code membership.gif"/>
Membership tiers can be locked so that only users who hold a specified promo code can purchase them.

<Steps>
    <Step title="Enable 'Invite Only'">
        In the target membership type's edit panel, enable "Invite Only" under "Advanced Options."
            <img src="/images/creating-campaign/membership/inviteonly/invite only toggle.png"/>
    </Step>
    <Step title="Create Code">
        <Info>
            For detailed instructions on how to create a promo code, please refer to our [tutorial here](/distribution/discountpromocode). Both manually and bulk generated codes can be created for invite-only memberships.
        </Info>
        Ensure that your code applies to the target membership type. Click "Add" and then "Update" in the sidebar when complete. The promo code is now activated.
        <img src="/images/creating-campaign/membership/inviteonly/creating code.gif"/>
        <Tip>
            To create a code that unlocks membership tiers but does not grant discounts, input '0' in the "Discount %" field.
        </Tip>
    </Step>
</Steps>
